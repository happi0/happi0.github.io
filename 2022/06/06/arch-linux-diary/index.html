change kernel on arch linux

  • download linux, linux-header
  • add it to grub
  • reboot

there are three kinds of kernel

  • old kernel
  • lts kernel, long-term support
  • zen kel, best version for users

there is tool for users to download out-of-date package

download it , and download linux kernel though it, make grub and reboot

next time,you can choose the kernel when in grub
